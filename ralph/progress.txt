## Codebase Patterns
- DialogData.ts 使用 DIALOGS 对象存储对话，格式为 { dialogId: { lines: [...] } }
- 每个对话至少需要 speaker 和 text 字段
- quests.json 使用数组存储任务，每个任务有 id, name, description, objectives, rewards 字段
- 仙剑对话风格：古风文雅，情感真挚，符合武侠仙侠氛围
- Map locations have: id, name, displayName, maps[], shops[], inns[], chapter?, description?
- Maps have: id, name, displayName, description, locationId, chapter, width, height, tileWidth, tileHeight, colorTheme, environment, layers[], npcs[], transitions[], events[], collisions[]
- DialogLine structure: speakerId, speakerName, text, expression (Expression enum), nextDialogId
- Expression values: NORMAL, HAPPY, SAD, ANGRY, SURPRISED, THINKING, SHY
