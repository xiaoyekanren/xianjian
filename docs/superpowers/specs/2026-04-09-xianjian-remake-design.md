# 仙剑奇侠传现代重制版 - 设计文档

## 项目概述

**项目名称：** 仙剑奇侠传现代重制版

**目标：** 保持原版剧情，使用现代Web技术重新设计画面和UI，制作完整重制版。

**技术栈：** HTML5 + TypeScript + Phaser.js

**美术风格：** 现代扁平化卡通风格，色彩明快，线条清晰

**战斗系统：** 经典回合制，界面和动画现代化，支持四人战斗

## 2. 战斗系统

### 战斗流程
```
战斗开始 → 显示敌人 → 玩家选择行动 → 执行行动 → 敌人行动 → 循环 → 战斗结束
```

### 玩家可选行动
- **攻击** - 物理攻击
- **仙术** - 使用魔法技能（消耗灵力）
- **道具** - 使用物品（药品等）
- **防御** - 减少受到伤害
- **逃跑** - 尝试脱离战斗（非BOSS战）

### 战斗数据结构
```typescript
interface BattleState {
  turn: number;
  playerParty: Character[];    // 最多4人
  enemies: Enemy[];
  currentActorIndex: number;
  actionQueue: Action[];
  comboReady: boolean;
  comboMembers: string[];
}

interface Action {
  type: 'attack' | 'skill' | 'item' | 'defend' | 'flee';
  source: Character | Enemy;
  target?: Character | Enemy;
  skillId?: string;
  itemId?: string;
}
```

### 伤害计算公式
- 物理伤害 = (攻击力 - 敌方防御) × 随机系数
- 仙术伤害 = 技能基础伤害 × (灵力/100) × 属性克制系数

### 四人战斗站位
```
敌方区域（上方）：
[敌人区域]

玩家区域（下方）：
角色1  角色2  角色3  角色4
李逍遥 灵儿  月如  阿奴
HP条  HP条  HP条  HP条

行动选择区：
攻击 | 仙术 | 道具 | 防御 | 逃跑
```

## 29. 开发策略：渐进式开发

### 开发阶段
1. **阶段一：核心框架+第一章**
   - 搭建项目架构
   - 实现核心系统（战斗、对话、地图）
   - 完成余杭镇篇内容
   - 验证可行性

2. **阶段二：扩展章节**
   - 添加苏州篇、白河村篇
   - 完善商店、旅馆等系统
   - 测试并调整

3. **阶段三：中期章节**
   - 完成将军坟、扬州、京城篇
   - 添加月如、更多角色

4. **阶段四：蜀山+锁妖塔**
   - 锁妖塔迷宫
   - 月如牺牲剧情
   - 高级仙术系统

5. **阶段五：十年前+苗族篇**
   - 时光回溯机制
   - 苗族地图
   - 阿奴角色
   - 五灵珠收集

6. **阶段六：最终决战+结局**
   - 拜月教总部
   - 多结局系统
   - 四人合体技

7. **阶段七：美术优化+音效**
   - AI生成美术资源
   - 音效添加
   - 特效完善

8. **阶段八：测试+优化**
   - 全面测试
   - 性能优化
   - 发布
